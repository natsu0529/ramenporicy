ラーメンレーダー プライバシーポリシー

制定日: 2025年9月30日
改定日: 2025年10月24日

「ラーメンレーダー」（以下、本アプリ）は、ユーザーの皆様のプライバシーを尊重し、個人情報保護の重要性を認識した上で、以下のプライバシーポリシー（以下、本ポリシー）を定めます。

1. 取得する情報とその利用目的

本アプリは、以下の情報を取得し、アプリの主要機能の提供および改善のために利用します。

1-1. ユーザーの位置情報

  • 取得する情報: 現在地情報（GPS情報、Wi-Fi情報などに基づく緯度経度情報）
  • 利用目的:
    • 現在地からラーメン店の距離を計算し、独自のランキングスコアを算出するため。
    • ユーザーの現在地周辺にあるラーメン店を検索し、地図上に表示するため。
  • 情報保存期間: ユーザーがアプリを起動している間のみ利用し、原則としてサーバー等に永続的に保存しません。

1-2. Google Maps Platform API 関連情報

  • 取得する情報: Google Maps Platform APIを経由して取得する、ラーメン店の評価、名称、座標、距離情報。
  • 利用目的: ラーメン店のランキング作成、およびGoogleマップへのナビゲーション連携のため。

1-3. Google AdMob による広告配信

  • 取得する情報: 広告識別子（IDFA/AAID）、デバイス情報（機種名、OSバージョン等）、広告閲覧履歴、アプリ内での広告表示回数・クリック情報など。
  • 利用目的:
    • ユーザーの興味・関心に基づくパーソナライズド広告を表示するため。
    • 広告配信の効果測定および不正クリック防止のため。
    • アプリ運営のための収益化。
  • 同意取得について: パーソナライズド広告の表示にあたっては、アプリ起動時にトラッキング許可を求め、ユーザーの同意を得た上でIDFA等を利用します。
  • 第三者提供: 本アプリは広告配信のため、Google LLC にデータを送信します。詳細は Googleの広告ポリシー をご参照ください。

2. 情報の第三者提供

本アプリは、法令に定める場合を除き、取得したユーザーの個人情報および位置情報を、ユーザーの同意なく第三者に提供することはありません。

ただし、本アプリの機能提供のため、以下の第三者サービスに情報を連携します。

  • Google Maps Platform API (Google LLC): 店舗情報および距離計算に利用します。
  • Google AdMob (Google LLC): 広告配信および効果測定に利用します。

3. ユーザーの選択と管理

  • ユーザーは、アプリ起動時のトラッキング許可ダイアログで「許可しない」を選択することにより、パーソナライズド広告を無効化できます。
  • iOSの場合、端末設定「設定 > プライバシーとセキュリティ > トラッキング」から、トラッキング許可をいつでも変更できます。

4. 免責事項

本アプリの利用は、ユーザーご自身の責任において行われるものとします。本アプリの利用により生じた損害、または本アプリから遷移するリンク先で生じたいかなる損害についても、当方は一切の責任を負いません。

5. プライバシーポリシーの変更

本ポリシーの内容は、法令改正やサービス内容の変更に伴い、ユーザーに通知することなく変更することができるものとします。変更後のプライバシーポリシーは、本ウェブサイトに掲載した時点から効力を生じるものとします。
